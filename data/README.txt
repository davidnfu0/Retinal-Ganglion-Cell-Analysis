- This folder contains all the data. 
- The results of the experiments are not in this repository. The results are the property of those who carried out the experiment, so they cannot be published at the moment.
- If the data are obtained, it's important to store them in the data folder so that the notebooks can access them. 
- They should have the corresponding names, which can be checked in the configuration file.
- There are several data files that are generated as the notebooks are run.
